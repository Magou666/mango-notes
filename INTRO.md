# 🥭 mango 便签

> 一款专为 macOS 设计的轻量桌面便签工具，让你的想法始终漂浮在桌面最前方。

---

## 这是什么

mango 是一款透明、无边框的桌面便签应用。它不藏在 Dock 里，不躲进系统菜单，而是直接浮在你的桌面上，随时可见、随时可记。

设计灵感来自"好用便签"和浏览器标签页管理——多条便签可以聚合在同一个窗口里，也可以随时拆出来独立显示，再拖回去合并，完全由你决定。

---

## 核心功能

### 📌 便签管理

- **新建 / 编辑**：点击底部 `✦` 新建便签，直接在便签内编辑，自动保存，无需手动点击保存
- **重命名**：右键侧边栏中的便签名称，可以自定义标题；不设置时自动取正文第一行作为标题
- **颜色主题**：8 种颜色方案（黄、粉、蓝、绿、紫、橙、青、暗黑），点击标题栏右侧色块切换
- **透明度**：拖动标题栏中的💧滑块，实时调整当前便签的透明度
- **置顶**：点击 🟢 按钮将便签固定在所有窗口最前方，再次点击取消

### 🏷 标签系统

- 每条便签可以添加多个自定义标签
- 点击底部 `🏷` 按钮打开标签面板，输入后回车即可添加
- 标签支持随时删除

### 🗑 回收站

- 删除便签采用**软删除**：长按底部 `🗑` 按钮（约 0.8 秒）将便签移入回收站，不立即永久删除
- 在管理面板中可以查看回收站内容，支持恢复或永久删除
- 支持一键清空回收站

## 窗口交互

mango 最有特色的部分是它的窗口管理方式，参考了浏览器标签页的"撕出与合并"体验。

### 侧边栏：多便签聚合

每个便签窗口左侧都有一条细细的侧边栏，展示当前窗口里所有便签的色块和标题缩写。

- **点击**侧边栏中的便签条目 → 切换到该便签
- **右键**侧边栏中的便签条目 → 重命名 / 新建 / 删除

### 拖拽分离：一条便签变成独立窗口

当一个窗口里有 **2 条及以上**便签时：

1. 从侧边栏**按住并拖动**某条便签
2. 将其**拖出窗口边界**后松手
3. 该便签立刻变成一个**独立的新窗口**，原窗口侧边栏中该条消失

> 💡 当窗口里只剩 1 条便签时，拖拽会被禁用，防止产生空窗口。

### 拖拽合并：独立窗口归队

将一个独立便签窗口的**标题栏**拖动，移到另一个便签窗口上方，使两者重叠：

- 重叠后约 **0.5 秒**，该便签自动合并进目标窗口的侧边栏
- 底部会出现绿色横幅提示「松开标题栏 → 合并到此窗口」，也可点击**立即合并**按钮手动触发

### 拖入合并：从侧边栏直接拖进另一窗口

从侧边栏拖出一条便签，**直接拖到另一个便签窗口上方后松手**：

- 如果鼠标落点在某个便签窗口内，便签直接合并进那个窗口，不会新建独立窗口
- 如果鼠标落在空白桌面，则正常新建独立窗口

## 管理面板

点击便签底部 `☰` 按钮，或托盘菜单中的「打开面板」，可以打开全局管理面板：

- 查看所有便签（包括回收站）
- 按标签筛选便签
- 恢复 / 永久删除回收站内容
- 清空回收站

---

## 快捷键与托盘

| 操作 | 方式 |
|------|------|
| 新建便签 | `⌘ + Shift + N`（全局快捷键，无需聚焦应用） |
| 新建便签 | 点击便签底部 `✦` 按钮 |
| 打开管理面板 | 点击便签底部 `☰` 按钮 |
| 打开管理面板 | 托盘菜单 → 「☰ 打开面板」 |
| 新建便签（托盘） | 托盘菜单 → 「🥭 新建便签」 |
| 退出应用 | 托盘菜单 → 「退出」 |

> mango 不会出现在 Dock 中，关闭所有窗口不会退出应用，通过托盘可以随时唤起。

## 数据存储

所有便签数据保存在本地，路径为：

```
~/Library/Application Support/mango/mango-notes/notes.json
```

纯 JSON 格式，可以直接备份。不联网，不上传任何数据。
