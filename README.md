# 🥭 mango

**一款专为 macOS 设计的轻量桌面便签，让想法始终浮在桌面最前方。**

透明无边框、多便签聚合、支持拖拽分离与合并——参考浏览器标签页的管理思路，带来更自由的便签空间组织体验。

---

## ✨ 功能亮点

- 🪟 **透明无边框窗口** — 完美融入桌面，不遮挡视线
- 📑 **侧边栏多便签聚合** — 多条便签共存于一个窗口，像浏览器标签页一样切换
- 🖱 **拖拽分离 / 合并** — 从侧边栏拖出一条便签变成独立窗口；拖回去自动合并
- 🎨 **8 种颜色主题** — 黄 / 粉 / 蓝 / 绿 / 紫 / 橙 / 青 / 暗黑，随手切换
- 💧 **实时透明度调节** — 标题栏滑块拖动，所见即所得
- 📌 **窗口置顶** — 钉在所有应用最前方，随时可见
- 🏷 **标签管理** — 每条便签支持多标签，管理面板支持按标签筛选
- 🗑 **软删除回收站** — 长按删除移入回收站，支持恢复，不怕误删
- ⌨️ **全局快捷键** — `⌘ Shift N` 随时新建，无需切换应用
- 💾 **本地存储** — 数据保存在本机 JSON 文件，不联网，不上传

## 🚀 安装与运行

### 直接使用（推荐）

从 [Releases](https://github.com/Magou666/mango-notes/releases) 下载最新的 `.dmg` 文件，拖入 Applications 即可。

### 本地开发

```bash
# 克隆项目
git clone https://github.com/Magou666/mango-notes.git
cd mango-notes

# 安装依赖
npm install

# 开发模式运行
npm start

# 打包构建（输出到 dist/）
npm run build
```

> **系统要求**：macOS，Apple Silicon（ARM64）

## 🖱 使用说明

### 基本操作

| 操作 | 方式 |
|------|------|
| 新建便签 | 底部 `✦` 按钮，或 `⌘ Shift N` 全局快捷键 |
| 切换便签 | 点击左侧侧边栏中的便签条目 |
| 重命名 | 右键侧边栏条目 → 重命名 |
| 删除便签 | 长按底部 `🗑` 约 0.8 秒，移入回收站 |
| 切换颜色 | 点击标题栏右侧色块 |
| 调整透明度 | 拖动标题栏 💧 滑块 |
| 置顶窗口 | 点击标题栏 🟢 按钮 |
| 打开管理面板 | 底部 `☰` 按钮，或托盘菜单 |

### 拖拽窗口管理

**分离**：当窗口里有 2 条及以上便签时，从侧边栏按住拖动某条，拖出窗口边界松手 → 独立新窗口

**合并（方式一）**：把独立窗口的标题栏拖动叠加到另一个便签窗口上，重叠 0.5 秒自动合并

**合并（方式二）**：从侧边栏拖出一条便签，直接拖到另一个便签窗口上方松手 → 直接合并进去

```
mango-notes/
├── src/
│   ├── main.js          # 主进程：窗口管理、IPC、数据持久化
│   ├── preload.js       # 渲染进程白名单 API
│   ├── assets/          # 图标资源
│   └── renderer/

│       └── panel.html   # 管理面板
├── package.json
└── INTRO.md             # 产品介绍文档
```

## 🛠 技术栈

- **Electron 28** — 跨平台桌面应用框架
- **Vanilla JS + HTML/CSS** — 无任何前端框架，轻量纯净
- **electron-builder** — 打包构建
- **本地 JSON** — 数据持久化，无数据库依赖
